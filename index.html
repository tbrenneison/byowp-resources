About Today's Workshop

Today, we will build a simple web page using HTML and CSS! This will take anywhere from 60 - 90 minutes.

At the end of the workshop, you'll have your own website that looks something like this one.

I'd like to also give a huge thank you to NASA Glenn Research Center for inviting me to do this workshop for Girls in ICT Day!

Agenda

5 minutes: introductions

5 minutes: content creation

30 minutes: creating a page with HTML

10 minute break

30 minutes: styling the page with CSS

15 minutes (optional): career Q & A session or other questions

What we Learnt in Today's Workshop

  • HTML (HyperText Markup Language) for creating content
  • CSS (Cascading Style Sheets) for styling content
  • Visual Studio Code text editor for writing code
